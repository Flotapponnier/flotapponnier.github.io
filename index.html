🌟 LET'S LEARN ANIMAL NAMES! 🌟

Farm Animals:

  • Cow = 牛 → Moo! Moo!
  • Horse = 马 → Neigh! Neigh!
  • Pig = 猪 → Oink! Oink!
  • Sheep = 羊 → Baa! Baa!
  • Chicken = 鸡 → Cluck! Cluck!

Wild Animals:

  • Lion = 狮子 → Roar! Roar!
  • Elephant = 大象 → Trumpet! Trumpet!
  • Monkey = 猴子 → Ooh! Ooh! Aah! Aah!
  • Giraffe = 长颈鹿
  • Zebra = 斑马

Pets:

  • Dog = 狗 → Woof! Woof!
  • Cat = 猫 → Meow! Meow!
  • Fish = 鱼 → Bubble! Bubble!
  • Rabbit = 兔子
  • Hamster = 仓鼠

📝 SIMPLE SENTENCES TO PRACTICE 📝

  • I have a dog. 我有一只狗。
  • I like cats. 我喜欢猫。
  • The lion is big. 狮子很大。
  • The mouse is small. 老鼠很小。
  • The bird can fly. 鸟会飞。
  • The fish can swim. 鱼会游泳。
